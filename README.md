# Features
- Generates random quotes
- Uses API integration (type.fit API)
- Prevents repeating the same quote consecutively
- Responsive and clean UI
- Fallback offline quotes if API fails
- Smooth hover effects and modern gradient design

# Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

# How it works
- User clicks the Get New Quote button.
- JavaScript fetches quotes from the API.
- A random quote is selected and displayed.
- If the API fails, local backup quotes are shown.
